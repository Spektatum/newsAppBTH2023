# News App BTH 2023
Project in vanilla JavaScript for BTH (https://www.bth.se) 
A-graded and marked 'Excellent'.

## Technology
JavaScript vanilla framework set up using composer and webpack. 
Loads dynamically, SPA single page application.

## Build 
Build with webpack and composer.json, npm. 
Webpack config file: webpack.config.js 
Starts in: js/app/app.js and with the bundle file in index.php.
